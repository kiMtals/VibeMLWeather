# 🌤 ML Прогноз Погоды (AI Weather Forecast)
Ради эксперимента ФУЛ навайбкоженно:)


Программа для предсказания погоды на 3 дня вперед с использованием машинного обучения и данных API Open-Meteo. Приложение анализирует погодные условия за последние 7 дней и выдает комплексный прогноз (температура, осадки, давление, ветер) через современный графический интерфейс.

## ⚙️ Основной функционал
* **Сбор данных:** Автоматическая загрузка исторических и актуальных данных о погоде через бесплатное API Open-Meteo (без необходимости API-ключей).
* **Машинное обучение:** Использование модели `RandomForestRegressor` для анализа временных рядов (метод "скользящего окна").
* **Графический интерфейс:** Современный и удобный интерфейс в темной теме, созданный с помощью `CustomTkinter`.

---

## 📂 Структура проекта

* `app.py` — Главный файл приложения с графическим интерфейсом. Отвечает за запрос свежих данных, запуск модели и красивый вывод прогноза.
* `train_model.py` — Скрипт для скачивания исторического архива погоды за несколько лет, разметки данных и обучения нейросети. *(Примечание: сохраните код обучения в этот отдельный файл)*.
* `weather_rf_model3.pkl` — Сохраненная обученная модель (появляется автоматически после запуска `train_model.py`).
* `req.txt` — Список необходимых Python-библиотек для работы проекта.

Для запуска проекта на вашем компьютере должен быть установлен **Python 3.8** или новее.

Скачайте файлы проекта в удобную для вас папку и откройте её в терминале (командной строке).
